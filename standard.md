# 用字標準說明

## 个、個、箇

| 字  | 辨析                 |
| --- | -------------------- |
| 个  | keh 助詞             |
| 個  | keh 量詞             |
| 個  | kóu 单独：～人，～性 |
| 箇  | kóu 指代词           |

## 着

| 讀音 | 辨析                                                         |
| ---- | ------------------------------------------------------------ |
| jiak | 表示狀態个持續或已有着結果：睏弗着。                         |
| tzek | 擺扗動詞後，表示動作个結束；擺扗句末或句中停頓處：飯喫過着。 |
| ciak | 動詞，着衣裳、着棋                                           |

>
